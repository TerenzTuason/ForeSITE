Core Principle

Our foundation for innovative education

Empowering Futures Thinking Education Through an Adaptive Learning System that personalizes learner experience using Hard Majority Voting for learning style assessment and K-Means Clustering for dynamic collaborative group formation.

Objectives

Driving innovation in adaptive learning

General Objective

To develop and evaluate an Adaptive Learning System (ALS) for Futures Thinking education which employs Hard Majority Voting for Honey and Mumford model learner style assessment and K-Means Clustering for collaborative group creation.

Accessible Online ALS

Design and develop an accessible online Adaptive Learning System with automatic content adaptation based on Honey and Mumford model learner style results.

Learning Modules Integration

Create and integrate various learning modules based on "The Futures Toolkit", catering to all four learning styles: Activist, Reflector, Theorist, and Pragmatist.

Learner Classification

Implement Hard Majority Voting with multiple algorithms to determine accurate learner classifications using the Honey and Mumford Learning Styles model.

Collaborative Team Building

Apply K-Means Clustering to create effective collaborative teams based on learner style results and user profiles.

System Evaluation

Conduct comprehensive evaluation of the ALS, including learner achievement assessment and algorithm performance analysis.
